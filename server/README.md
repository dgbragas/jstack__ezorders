# **ezOrders: Server**

Server constructed using MondoDB with Mongoose to provide an API to mobile and web applications.

## **:heavy_plus_sign: Principle functionalities**

- List all inserted orders
- Insert a new order
- Update order status

## **:fire: Principle libraries**

- **cors:** v2.8.5
- **express:** v4.17.1
- **mongoose:** v5.11.15
- **socket.io:** v3.1.1
